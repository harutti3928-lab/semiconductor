# 半導体物理学 まとめノート

大学院講義「**半導体物理学**」（令和8年度・全11回）の内容を自分の復習用にまとめた
ノート集です。真性半導体のキャリア統計から、pn接合・各種デバイス、電子輸送、
2次元電子系・量子ホール効果・メゾスコピック伝導まで、半導体物理の主要トピックを
数式の導出込みで整理しています。

> **数式表示について**
> このノートは GitHub の数式レンダリング（LaTeX / MathJax）を利用しています。
> インライン数式は `$...$`、別行立て数式は `$$...$$` で記述しており、
> GitHub 上でそのまま数式として表示されます。
