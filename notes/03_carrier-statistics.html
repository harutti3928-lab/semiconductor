第3回　真性半導体と外因性半導体のキャリア統計

— 状態密度・質量作用の法則・キャリア密度の温度依存性 —

この回の内容

  • 真性半導体（固有半導体）
    • エネルギーバンド内でのキャリア（電子・正孔）分布
    • 質量作用の法則、\(pn\) 積
  • 外因性半導体（不純物半導体）
    • ドナーとアクセプタ
    • \(n\) 型半導体のキャリア分布（ドナー準位が1つ／補償あり）
    • 縮退半導体

1. 半導体とは

電気伝導率 \(\sigma\) による分類の目安は

\[10^{-10}\ \mathrm{S/m} < \sigma_{\text{半導体}} < 10^{5}\ \mathrm{S/m}\]

であり、絶縁体と金属の中間に位置する。ただし \(\sigma\) の値だけで金属・半導体・絶縁体を区別するのは不正確で、本質的な違いは \(T=0\) でのバンド構造 にある。

  • 金属：フェルミ準位 \(E_F\) が許容帯の中にある（部分的に満たされたバンドがある）。
  • 半導体・絶縁体：\(T=0\) で価電子帯が完全に満たされ、伝導帯が空。両者の間に禁制帯（バンドギャップ \(E_g\)）がある。半導体はギャップが比較的小さく、絶縁体は大きい。

半導体の特徴は次の2点に集約される。

  1. 電気伝導率 \(\sigma\) を 制御できる（ドーピング、熱、光、電場、磁場などでキャリア数を変える）。
  2. 禁制帯幅 \(E_g\) が材料によって異なる。

本章の目標は、キャリア密度とフェルミ準位 \(E_F\) の温度依存性を求めることである。

2. 真性半導体のキャリア

真性半導体（intrinsic semiconductor）は、伝導に関与するキャリアを放出する不純物原子を含まない半導体である。キャリアは熱的励起によって生じ、

\[\text{伝導帯内の自由電子} \;+\; \text{価電子帯内の自由正孔}\]

が必ず対で生成される。記号を次のように定める。

記号意味
\(E_C\)伝導帯の底（最低エネルギー）
\(E_V\)価電子帯の頂上（最高エネルギー）
\(E_g = E_C - E_V\)禁制帯幅（バンドギャップ）
\(E_i\)真性半導体のフェルミ準位

3. 伝導帯内の電子統計

モデル

伝導帯の底と価電子帯の頂上がともに \(k=0\) にあり、\(E\)–\(k\) 関係が放物線的であると仮定する。

\[E = E_C + \frac{\hbar^2 k^2}{2m_e}, \qquad E = E_V - \frac{\hbar^2 k^2}{2m_h}\]

ここで \(m_e,\ m_h\) は伝導電子・正孔の有効質量である。

電子密度の積分表式

伝導帯内の電子密度 \(n\) は、状態密度 \(N_e(E)\) とフェルミ・ディラック分布 \(f_e(E)\) の積を伝導帯全体で積分して得られる。

\[n = \int_{E_C}^{\infty} N_e(E)\, f_e(E)\, dE, \qquad f_e(E) = \frac{1}{1+\exp\!\left[(E-E_F)/k_BT\right]}\]

放物線バンドの状態密度は

\[N_e(E) = \frac{1}{2\pi^2}\left(\frac{2m_e}{\hbar^2}\right)^{3/2}\sqrt{E-E_C}\;\propto\; m_e^{3/2}\sqrt{E-E_C}.\]

ボルツマン近似

\(E-E_F \gg k_BT\)（非縮退）のとき、フェルミ・ディラック分布はボルツマン分布で近似できる。

\[f_e(E) \simeq \exp\!\left[-\frac{E-E_F}{k_BT}\right]\]

これを代入し、\(E'=E-E_C\) と置換すると

\[n = \frac{1}{2\pi^2}\left(\frac{2m_e}{\hbar^2}\right)^{3/2} \exp\!\left[-\frac{E_C-E_F}{k_BT}\right]\int_0^{\infty}\sqrt{E'}\,e^{-E'/k_BT}\,dE'.\]

ガウス積分 \(\int_0^\infty \sqrt{E'}\,e^{-E'/k_BT}dE' = \tfrac{\sqrt\pi}{2}(k_BT)^{3/2}\) を用いると、

\[\boxed{\,n(T) = N_C(T)\,\exp\!\left[-\frac{E_C-E_F}{k_BT}\right]\,} \qquad N_C(T) = 2\left(\frac{2\pi m_e k_BT}{h^2}\right)^{3/2}\]

\(N_C\) を 伝導帯の有効状態密度という。これは「伝導帯の底 \(E_C\) に状態密度 \(N_C\) が集中しているとみなし、そこにボルツマン因子を掛けると電子密度が得られる」と解釈できる。

4. 価電子帯内の正孔統計

正孔密度 \(p\) は、価電子帯内で電子が占有されていない確率 \(1-f_e(E)\) を用いて

\[p = \int_{-\infty}^{E_V} N_h(E)\bigl[1-f_e(E)\bigr]\,dE.\]

同様のボルツマン近似で

\[\boxed{\,p(T) = N_V(T)\,\exp\!\left[-\frac{E_F-E_V}{k_BT}\right]\,} \qquad N_V(T) = 2\left(\frac{2\pi m_h k_BT}{h^2}\right)^{3/2}\]

\(N_V\) を 価電子帯の有効状態密度という。

5. 真性半導体のフェルミ準位 Ei

真性条件 \(n = p \equiv n_i\) を課すと、

\[N_C\exp\!\left[-\frac{E_C-E_F}{k_BT}\right] = N_V\exp\!\left[-\frac{E_F-E_V}{k_BT}\right].\]

両辺の対数をとって \(E_F\) について解くと、

\[E_F = E_i = \frac{E_C+E_V}{2} + \frac{k_BT}{2}\ln\frac{N_V}{N_C} = \frac{E_C+E_V}{2} + \frac{3k_BT}{4}\ln\frac{m_h}{m_e}.\]

第2の等号では \(N_V/N_C = (m_h/m_e)^{3/2}\) を用いた。真性フェルミ準位はほぼバンドギャップの中央にあり、\(m_h \ne m_e\) による温度依存のずれは小さい。

6. 質量作用の法則と pn 積

\(n\) と \(p\) の積をとると \(E_F\) が消え、

\[p\,n = N_V N_C\exp\!\left[-\frac{E_C-E_V}{k_BT}\right] = N_V N_C\exp\!\left[-\frac{E_g}{k_BT}\right].\]

真性半導体では \(n=p=n_i\) だから、

\[\boxed{\,n_i(T) = (N_V N_C)^{1/2}\exp\!\left[-\frac{E_g}{2k_BT}\right] = 2\left(\frac{2\pi k_BT}{h^2}\right)^{3/2}(m_e m_h)^{3/4}\exp\!\left[-\frac{E_g}{2k_BT}\right]\,}\]

したがって 質量作用の法則

\[\boxed{\,pn = n_i^2\,}\]

が成り立つ。これは熱平衡状態であれば真性・外因性を問わず成立する重要な関係である。

7. 外因性半導体：n 型と p 型

外因性半導体（extrinsic semiconductor）は、キャリアを放出する不純物原子（ドーパント）を含む半導体である。

型ドーパント条件多数キャリア少数キャリア
\(n\) 型\(N_D > N_A\)自由電子自由正孔
\(p\) 型\(N_D < N_A\)自由正孔自由電子
  • ドナー（donor）：伝導帯に電子を供給する置換型不純物。IV族半導体なら V族元素（Si に対する P, As など）。
  • アクセプタ（acceptor）：価電子帯に正孔を供給する（価電子帯の電子を捕獲する）置換型不純物。IV族半導体なら III族元素（Si に対する B など）。

8. 不純物準位：水素原子様モデル

ドナーは「1価のイオン核（\(+e\) の固定電荷）＋束縛電子」とみなせる。比誘電率 \(\varepsilon_r\) の媒質中、有効質量 \(m^*\) の電子に対するシュレーディンガー方程式

\[\left[-\frac{\hbar^2}{2m^*}\nabla^2 - \frac{e^2}{4\pi\varepsilon_0\varepsilon_r r}\right]\Psi(\mathbf r) = E\,\Psi(\mathbf r)\]

は水素原子と同型で、束縛エネルギーと有効ボーア半径は水素の値をスケールしたものになる。

\[E_n = -13.6\ \mathrm{eV}\cdot\frac{m^*}{m}\frac{1}{\varepsilon_r^2}\frac{1}{n^2}, \qquad a_B^* = a_B\,\frac{m}{m^*}\,\varepsilon_r = 0.053\ \mathrm{nm}\cdot\frac{m}{m^*}\varepsilon_r\]

Si の例（\(\varepsilon_r = 11.9,\ m^*=0.33m\)）では基底状態 \(E_1 \approx -32\ \mathrm{meV}\)、\(a_B^*\approx 1.9\ \mathrm{nm}\)。イオン化エネルギーが室温の \(k_BT(300\,\mathrm K)=25.9\ \mathrm{meV}\) と同程度に小さいため、室温では大半のドナーがイオン化する。

主なイオン化エネルギー（meV）：Si 中の P=44, As=49, B=45／GaAs 中の浅い準位は 6 前後。

9. ドナー準位の電子占有数

熱平衡でのドナー準位の占有を、次を仮定して求める。(1) 1つのドナーは1つの準位を作る、(2) 1準位に捕獲される電子は最大1個、(3) 異なるドナー間の電子相互作用はない。

グランドカノニカル分布で平均粒子数を評価すると、スピン縮退による因子2が現れ、中性ドナー密度 \(N_D^0\) とイオン化ドナー密度 \(N_D^+\) は

\[N_D^0 = N_D\,\frac{1}{1+\tfrac12\exp\!\left[(E_D-E_F)/k_BT\right]}, \qquad N_D^+ = N_D\,\frac{1}{1+2\exp\!\left[-(E_D-E_F)/k_BT\right]}.\]

アクセプタも同様に（正孔について因子は状況により異なるが）

\[N_A^- = N_A\,\frac{1}{1+2\exp\!\left[(E_A-E_F)/k_BT\right]}.\]

10. n 型半導体のキャリア密度の温度依存性

電気的中性条件

\[n(T) = p(T) + N_D^+(T)\]

に \(N_D^+\) と \(n,p\) の表式を代入すると \(E_F\) について解析的には解けない。そこで3つの温度領域に分けて考える。

(1) 高温・真性領域（ni ≫ ND）

ドナーは全てイオン化し、価電子帯からの熱励起がドナー由来の電子数を上回る。真性半導体に近づき、

\[n = p = n_i = (N_V N_C)^{1/2}\exp\!\left[-\frac{E_g}{2k_BT}\right], \qquad E_F \simeq E_i = \frac{E_g}{2}.\]

(2) 中間温度・出払（飽和）領域（ND ≫ ni）

ドナーはほぼ全てイオン化し、\(p\simeq 0\)。

\[n(T) \simeq N_D, \qquad p(T)\simeq \frac{n_i^2}{N_D}\ \text{（質量作用の法則）}.\]

フェルミ準位は

\[E_C - E_F = k_BT\ln\frac{N_C(T)}{N_D}, \qquad E_F - E_i = k_BT\ln\frac{N_D}{n_i}.\]

デバイス動作の基本となる領域で、キャリア密度がドナー密度で決まり温度に鈍感（プラトー）。

(3) 低温・不純物（凍結）領域（freeze-out）

価電子帯からの励起は無視でき、ドナー準位が電子を再捕獲し始める（\(E_F-E_D\gg k_BT\)）。中性条件と式 (9) の積から \(E_F\) を消去して

\[n(T) \simeq \left(\frac{N_D N_C(T)}{2}\right)^{1/2}\exp\!\left[-\frac{\Delta E_D}{2k_BT}\right], \qquad \Delta E_D \equiv E_C - E_D,\] \[E_F = \frac{E_C+E_D}{2} + \frac{k_BT}{2}\ln\frac{N_D}{2N_C}.\]

活性化エネルギーが \(\Delta E_D/2\) となる点が特徴（真性領域の \(E_g/2\) と対照的）。

まとめ図（模式）

\(\ln n\) を \(1/T\) に対してプロットすると、高温側から順に「真性領域（傾き \(-E_g/2\)）→ 出払領域（ほぼ平坦、\(n\simeq N_D\)）→ 凍結領域（傾き \(-\Delta E_D/2\)）」の3領域が現れる。

11. 補償のある場合（ND ≫ NA の n 型）

ドナーとアクセプタが共存すると、アクセプタはドナー電子を受け取って完全にイオン化する。電気的中性条件は

\[n(T) + N_A^- = p(T) + N_D^+.\]

極低温（\(N_A > n_{D\Rightarrow C}\)）では新たに 極低温・不純物領域 が現れ、

\[n(T) \simeq \frac{N_D - N_A}{N_A}\,\frac{N_C}{2}\exp\!\left[-\frac{\Delta E_D}{k_BT}\right], \qquad E_F = E_D + k_BT\ln\frac{N_D-N_A}{2N_A}.\]

活性化エネルギーが \(\Delta E_D/2\) ではなく \(\Delta E_D\) になる点が非補償の場合との違い。

12. 縮退半導体

高密度にドープして不純物密度が有効状態密度を超える（\(N_D,\,N_A > N_C,\,N_V\)、目安として \(>10^{18}\text{–}10^{19}\ \mathrm{cm^{-3}}\)）と、

\[E_C - E_F = k_BT\ln\frac{N_C(T)}{N_D} < 0\]

となり、フェルミ準位が伝導帯（\(n^+\)型）または価電子帯（\(p^+\)型）の中に入る。この状態を 縮退半導体という。

  • ドナー（アクセプタ）準位が広がってバンドを形成し、イオン化エネルギーが減少。
  • 不純物バンドが伝導帯（価電子帯）と合体し、イオン化エネルギーが実質 0 に。
  • ボルツマン近似が使えず、フェルミ・ディラック統計をそのまま扱う必要がある。

\(10^{18}\ \mathrm{cm^{-3}}\) は「一辺 10 nm の立方体中にドーパント1個」に相当し、有効ボーア半径（数 nm）と波動関数が重なり始めるスケールである。

キーポイント

  • 有効状態密度 \(N_C, N_V \propto T^{3/2}\) を用いると、キャリア密度は \(n=N_C e^{-(E_C-E_F)/k_BT}\)、\(p=N_V e^{-(E_F-E_V)/k_BT}\) とコンパクトに書ける。
  • 熱平衡では常に \(pn=n_i^2\)（質量作用の法則）。
  • \(n\) 型の \(\ln n\)–\(1/T\) 特性は「真性 → 出払 → 凍結」の3領域。出払領域で \(n\simeq N_D\)。
  • ドナー占有には因子2（スピン縮退）が入る。
  • 高濃度ドープでフェルミ準位がバンド内に入ると縮退半導体。
