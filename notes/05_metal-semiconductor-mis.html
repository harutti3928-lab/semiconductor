第5回　金属–半導体接合と MIS 構造

— ショットキー障壁・オーミック接触・MIS構造 —

この回の内容

  • 仕事関数と電子親和力
  • 理想的な金属–半導体接触：ショットキー障壁、I–V特性、オーミック接触
  • 理想的な金属/絶縁体/半導体（MIS）構造：蓄積・空乏・反転、表面電荷、二次元電子ガス
  • 金属/半導体界面のバンド配置：Schottky–Mott limit、フェルミ準位ピニング、Bardeen limit
  • 絶縁体/半導体界面のバンド配置とピニング

1. 仕事関数と電子親和力

  • 仕事関数 \(\phi\)：電子を物質表面から真空中に取り出すのに必要なエネルギー。
  • 電子親和力 \(\chi\)：真空の遠方から電子を物質に与えたときに得られる最大エネルギー（真空準位から \(E_C\) までの深さ）。

金属では仕事関数と電子親和力が一致するが、半導体・絶縁体では一致しない（\(E_C\) と \(E_F\) の間にギャップがあるため）。

2. 理想的な金属–半導体接触

仮定：理想界面、表面準位（界面準位）なし。\(n\) 型半導体（\(\phi_n\equiv E_C-E_F\)）を考える。

ショットキー障壁の形成（φM > χS + φn）

金属の仕事関数が大きいと、接触時に電子が半導体から金属へ移動し、半導体側にイオン化ドナーの空間電荷層ができる。金属側から見た障壁高さ（Schottky–Mott limit）は

\[\phi_{Bn} = \phi_M - \chi_S \quad(\text{電子に対する障壁}),\qquad \phi_{Bp} = \chi_S + E_g - \phi_M \quad(\text{正孔に対する障壁}).\]

空乏層とその容量

半導体側の空間電荷層厚さ \(d\) は、ポアソン方程式 \(d^2V/dx^2 = -eN_D/\varepsilon_s\) を境界条件のもとで解いて

\[d = \sqrt{\frac{2\varepsilon_s V_D}{eN_D}},\qquad eV_D = \phi_{Bn}-\phi_n\ (\text{半導体側の拡散電位}).\]

電圧 \(V\) 印加時は \(V_D\to V_D-V\) となり、

\[d = \sqrt{\frac{2\varepsilon_s(V_D-V)}{eN_D}},\qquad C = \frac{dQ}{dV} = \frac{\varepsilon_s}{d} = \sqrt{\frac{e\varepsilon_s N_D}{2(V_D-V)}}.\]

I–V 特性：整流作用（定性）

ショットキー接合は多数キャリアが主役。

  • 金属→半導体の電子流：障壁 \(\phi_{Bn}\) は印加電圧によらず一定。
  • 半導体→金属の電子流：障壁 \(e(V_D-V)\) は電圧に依存。

順バイアスで電流が流れ、逆バイアスでは小さな値に飽和する（整流作用）。

熱電子放出理論（Bethe 理論）

空乏層が薄い場合、電子が障壁を越える過程が支配的（Thermionic emission）。半導体→金属の電流密度を、障壁を越えられる最小速度 \(v_{0x}\)（\(\tfrac12 m^* v_{0x}^2 = e(V_D-V)\)）以上について積分すると、

\[J_{S\to M} = A^*T^2\exp\!\left(-\frac{\phi_B-eV}{k_BT}\right),\qquad A^* = \frac{4\pi e m^* k_B^2}{h^3}\ (\text{リチャードソン定数}).\]

金属→半導体の電流は電圧によらず \(J_{M\to S}=A^*T^2\exp(-\phi_B/k_BT)\)。正味の電流は

\[\boxed{\,J = J_{S\to M}-J_{M\to S} = A^*T^2\exp\!\left(-\frac{\phi_B}{k_BT}\right)\left[\exp\!\left(\frac{eV}{k_BT}\right)-1\right] = J_0\left[\exp\!\left(\frac{eV}{k_BT}\right)-1\right]\,}\]

pn接合と同じ整流特性だが、係数がリチャードソン定数を含む点、少数キャリアではなく多数キャリアが担う点が異なる。空乏層が厚い場合は Schottky 理論（空乏層内散乱を考慮）が必要。

オーミック接触（φM < χS + φn）

金属の仕事関数が小さいと、電子が金属から半導体へ移動して障壁ができず、両方向に電流が流れるオーミック接触になる。

接触\(n\) 型\(p\) 型
ショットキー\(\phi_M > \chi_S+\phi_n\)\(\phi_M < \chi_S+E_g+\phi_p\)
オーミック\(\phi_M < \chi_S+\phi_n\)\(\phi_M > \chi_S+E_g+\phi_p\)
p型ワイドギャップ半導体ではオーミック接触が取りにくい（正孔に対する障壁 \(\phi_{Bp}=\chi_S+E_g-\phi_M\) が \(E_g\) の大きさゆえ高くなるため）。

3. 理想 MIS 構造

金属/絶縁体/半導体（MIS）構造。仮定：理想界面、界面準位なし、絶縁体抵抗無限大（直流でキャリアは通れない）。

フラットバンド条件

熱平衡で半導体バンドが平坦になる条件（\(n\) 型）は

\[\phi_M = \chi_S + \phi_n = \chi_S + \frac{E_g}{2} - e\phi_B,\qquad e\phi_B \equiv E_F - E_i.\]

蓄積・空乏・反転（n 型半導体の場合）

金属に電圧 \(V\) を印加し、界面での表面電位 \(\psi_s = (E_{i}^{\text{bulk}}-E_i^{\text{surf}})/e\) で状態を分類する。界面での密度は

\[n_s = n_{n0}\exp\!\left(\frac{e\psi_s}{k_BT}\right),\qquad p_s = p_{n0}\exp\!\left(-\frac{e\psi_s}{k_BT}\right).\]
状態条件バンドの曲がり
蓄積（accumulation）\(\psi_s>0\) (\(V>0\))下向き、多数電子が界面に集まる
フラットバンド\(\psi_s=0\)平坦
空乏（depletion）\(\phi_B<\psi_s<0\)上向き、電子が空乏化
真性\(\psi_s=\phi_B\)\(n_s=p_s=n_i\)
反転（inversion）\(\psi_s<\phi_B\) (\(V<0\))上向き、界面に正孔が溜まり表面が p 型化

表面電荷（ポアソン方程式）

界面に垂直方向のポアソン方程式 \(d^2\psi/dx^2 = -\rho(x)/\varepsilon_s\) に \(d\psi/dx\) を掛けて積分すると、電場 \(\mathcal E=-d\psi/dx\) と表面電位の関係が得られる。\(x=0\) で \(\psi=\psi_s\)、\(\mathcal E=\mathcal E_s\) とすると、半導体内に誘起される単位面積あたりの総電荷は

\[Q_S = \varepsilon_s\mathcal E_s = \pm\sqrt{2\varepsilon_s k_BT N_D}\left[\frac{n_i^2}{N_D^2}\bigl(e^{-e\psi_s/k_BT}+\tfrac{e\psi_s}{k_BT}-1\bigr)+\bigl(e^{e\psi_s/k_BT}-\tfrac{e\psi_s}{k_BT}-1\bigr)\right]^{1/2}.\]

\(|Q_S|\) を \(\psi_s\) に対して両対数プロットすると、蓄積・弱反転・強反転で \(\sim e^{|e\psi_s|/2k_BT}\)、空乏で \(\sim\sqrt{\psi_s}\) と傾きが変わる。

強反転条件

反転層の正孔密度が界面で \(N_D\) に達する条件から、強反転を与える表面電位は

\[\boxed{\,-\psi_s(\text{inv}) = 2\phi_B = \frac{2k_BT}{e}\ln\frac{N_D}{n_i}\,}\]

これが MOSFET のしきい値を決める基本条件になる。

4. MIS 反転層と二次元電子ガス

反転層の界面垂直方向の幅が電子の平均自由行程より狭くなると量子効果が現れる。\(p\) 型半導体の反転層では、界面垂直方向に三角ポテンシャル \(eE_G z\)（\(E_G\)：ゲート電場）で電子が閉じ込められ、サブバンドが形成される。

簡単なド・ブロイ波長評価（\(z_n = n\lambda_z/2\)）から

\[E_{z,n} = \frac{1}{(2m_\perp)^{1/3}}(e E_G)^{2/3}\,c_n^{2/3}\quad(n=1,2,\dots)\]

厳密には三角ポテンシャルの解はエアリー関数で表され、

\[E_{z,n} = \frac{1}{(2m_\perp)^{1/3}}(e E_G)^{2/3}\,\alpha_n,\qquad \alpha_0=2.338,\ \alpha_1=4.087,\ \alpha_2=5.520,\dots\]

Si の例（\(m_\perp=0.25m\)、\(E_G=6.7\times10^5\ \mathrm{V/cm}\)）では \(E_{z,1}\approx186\ \mathrm{meV}\)、\(z_1\approx2.8\ \mathrm{nm}\)。室温でも量子化は無視できず、界面に二次元電子ガス（2DEG）が形成される。

5. 金属/半導体界面のバンド配置とピニング

Schottky–Mott モデルの破綻

Schottky–Mott limit（\(\phi_{Bn}=\phi_M-\chi_S\)）は整流性は説明できるが、定量的に実験と全く合わない。多くの場合、障壁高さは金属の仕事関数にほとんど依存しない。

フェルミ準位ピニング

界面で半導体側のフェルミ準位が、バンドギャップ内の特徴的エネルギー \(E_0\) 近傍に固定される現象をフェルミ準位ピニングという。

  • Bardeen の考え：界面の真性局在準位がショットキー障壁をピン留めする。
  • Si には真性表面準位があるが、GaAs(110), InP(110) には真性表面準位がないのにピニングが起こり、しかも金属の種類によらず同じ準位にピン留めされる → 別の機構が必要。
少数の表面準位でも十分ピン留めできる：Si(111) 理想表面のダングリングボンド密度は \(\sim 7.8\times10^{14}\ \mathrm{cm^{-2}}\)。これはキャリア密度 \(1\times10^{18}\ \mathrm{cm^{-3}}\) の Si でいえば厚さ \(7.8\ \mathrm{\mu m}\) 分のキャリア数に相当する（\(7.8\times10^{14}=1\times10^{18}\times t\)）。ごく薄い界面層の準位でもバンドを曲げ、フェルミ準位を固定するのに足りる電荷を蓄えうる。

S パラメータ

障壁高さの金属仕事関数依存性を

\[S = \frac{d\phi_{Bn}}{d\phi_M}\]

で表す。\(S=1\) が Schottky–Mott limit、\(S=0\) が Bardeen limit（完全ピニング）。

表面準位とギャップ内準位のモデル

  • 表面準位（surface states）：表面近傍に局在する電子状態。
    • タム状態：表面近傍のポテンシャルの違いで生じる。
    • ショックレー状態：バルクでギャップを作っていた要因（ブラッグ干渉、結合・反結合）が表面で失われて生じる。ダングリングボンドもこの一種。
  • ギャップ内準位の帯電：acceptor-like（空で中性・占有で負、電子トラップ）、donor-like（空で正・占有で中性、正孔トラップ）。一般に acceptor-like が上にある。
  • 各種モデル：
    • MIGS（metal induced gap state）：金属波動関数の裾がギャップ内にエバネセント波として侵入し界面準位となる。
    • UDM（統一欠陥モデル）：金属付着時に表面に空孔・アンチサイトなどの欠陥ができ、離散準位がピニングを起こす。
    • DIGS（disorder induced gap state）：界面形成でボンドが乱れ、ギャップ内に連続分布するドナー型・アクセプタ型状態ができる。

6. 絶縁体/半導体界面のバンド配置とピニング

  • ギャップ内に界面準位があると、絶縁体/半導体界面でもフェルミ準位ピニングが起こる。
  • 熱酸化 SiO₂/Si 界面は界面準位を十分低減でき、MOSFET の実用化に大きく貢献した。
  • 化合物半導体は良質な絶縁体/半導体界面の制御が格段に難しく、いまだ MOSFET の実用化に至っていない（→ 第9回 III-V MOSFET の課題につながる）。

キーポイント

  • ショットキー障壁 \(\phi_{Bn}=\phi_M-\chi_S\)（理想）、熱電子放出で \(J=A^*T^2 e^{-\phi_B/k_BT}(e^{eV/k_BT}-1)\)。多数キャリア支配。
  • オーミック接触は仕事関数条件で決まり、p型ワイドギャップで取りにくい。
  • MIS の強反転条件 \(-\psi_s=2\phi_B\) が MOSFET しきい値の基礎。
  • 反転層は三角ポテンシャル閉じ込めで 2DEG（エアリー関数、サブバンド）。
  • 実際の障壁高さは仕事関数に鈍感 → フェルミ準位ピニング（\(S\) パラメータ、MIGS/UDM/DIGS）。
  • SiO₂/Si は低界面準位で MOSFET を可能にした。
