# 第4回 pn接合

## この回の内容

- pn接合の作製方法
- エネルギーダイアグラム（熱平衡状態）
- pn接合の1次元モデル（ポアソン方程式）
- キャリアの生成と再結合
- pn接合の電流–電圧（I–V）特性

---

## 1. pn接合とその作り方

**pn接合**とは、1つの半導体結晶内に $n$ 型領域と $p$ 型領域を密着させたものである。作製方法には次がある。

- 不純物金属との合金を作る
- 不純物を拡散させる
- 不純物をイオン打ち込みする
- エピタキシャル成長させる

---

## 2. 熱平衡状態のエネルギーダイアグラム

同じ半導体（例：Si）の pn 接合では $E_{Cn}=E_{Cp}=E_C$、$E_{Vn}=E_{Vp}=E_V$ だが、接合前は $n$ 型と $p$ 型でフェルミ準位が異なる。接合前の各側では

$$E_{Fn} - E_i = k_BT\ln\frac{N_D}{n_i}, \qquad E_i - E_{Fp} = k_BT\ln\frac{N_A}{n_i}.$$

**接合直後に起こること**

1. $n$ 型・$p$ 型でキャリア密度に勾配があるため、電子は $n\to p$、正孔は $p\to n$ へ**拡散**する。
2. 拡散したキャリアは逆型領域の多数キャリアと**再結合**し、接合近傍にキャリアのいない領域＝**空乏層（depletion layer）**ができる。
3. 空乏層にはイオン化したドナー（$+$）とアクセプタ（$-$）が残り、**空間電荷層（space charge region）**を形成。これが作る電場 $\mathcal E$ が拡散をちょうど止める。

熱平衡ではフェルミ準位が全体で一致し、バンドが曲がる。この曲がりの電位差が**ビルトインポテンシャル（内蔵電位）**$V_{bi}$ である。

---

## 3. ビルトインポテンシャルの導出

接合形成で生じたポテンシャル $\phi(x)$ を考え、$E_C, E_V$ を $-e\phi(x)$ だけシフトさせると、キャリア密度は

$$n(x) = N_C\exp\!\left[-\frac{(E_C-e\phi(x))-E_F}{k_BT}\right],\qquad
p(x) = N_V\exp\!\left[-\frac{E_F-(E_V-e\phi(x))}{k_BT}\right].$$

接合から十分離れた中性領域では $N_D=n(-\infty)$、$N_A=p(\infty)$ が成り立つ。両者の積をとって整理すると、

$$\boxed{\,eV_{bi}=e\bigl[\phi(-\infty)-\phi(\infty)\bigr]
= E_g - k_BT\ln\frac{N_C N_V}{N_D N_A}
= k_BT\ln\frac{N_D N_A}{n_i^2}\,}$$

各側の寄与に分けると、

$$eV_n = k_BT\ln\frac{N_D}{n_i},\qquad eV_p = k_BT\ln\frac{N_A}{n_i},\qquad V_{bi}=V_n+V_p.$$

---

## 4. 1次元モデル（ポアソン方程式）

### モデルの仮定

1. ドーパントは一様分布、1次元で扱う。
2. $n$/$p$ 界面は急峻。
3. ドーパントはすべて活性化。
4. 空乏層は完全に空乏化（**空乏近似**）。

### ポアソン方程式と電場

空乏近似のもとで、$-x_n<x<0$（$n$ 側）と $0<x<x_p$（$p$ 側）で

$$\frac{d^2V}{dx^2} = -\frac{q}{\varepsilon_s}N_D \ (n\text{側}),\qquad
\frac{d^2V}{dx^2} = +\frac{q}{\varepsilon_s}N_A \ (p\text{側}).$$

空乏層端で電場ゼロという境界条件で積分すると、電場は接合面 $x=0$ で最大となる三角形分布になる。

$$\mathcal E_{\max} = \frac{q}{\varepsilon_s}N_D x_n = \frac{q}{\varepsilon_s}N_A x_p.$$

この式は **電荷中性条件** $N_D x_n = N_A x_p$ を含む（$n$ 側と $p$ 側の空間電荷量が等しい）。

### ビルトインポテンシャルと空乏層幅

電場をもう一度積分すると電位が求まり、

$$V_{bi} = \frac{q}{2\varepsilon_s}\bigl(N_D x_n^2 + N_A x_p^2\bigr) = \frac{\mathcal E_{\max}}{2}(x_n+x_p).$$

$N_D x_n=N_A x_p$ と組み合わせて空乏層幅 $W=x_n+x_p$ を解くと、

$$\boxed{\,W = \sqrt{\frac{2\varepsilon_s}{q}\,\frac{N_A+N_D}{N_A N_D}\,V_{bi}}\,}$$

$$x_n = \sqrt{\frac{2\varepsilon_s}{q}\frac{1}{N_A+N_D}\frac{N_A}{N_D}V_{bi}},\qquad
x_n:x_p = N_A:N_D.$$

低ドープ側に空乏層が広く伸びる（片側階段接合の基礎）。

---

## 5. 外部電圧印加時：空乏層と接合容量

$p$ 側の電位を $n$ 側に対して下げる向きを**順方向（正）**にとる。外部電圧 $V$ はほぼ全て抵抗の高い空乏層にかかるので、接合部の実効電位は $V_{bi}-V$ となる。空乏層幅は

$$W(V) = \sqrt{\frac{2\varepsilon_s}{q}\frac{N_A+N_D}{N_A N_D}(V_{bi}-V)} = W(0)\sqrt{1-\frac{V}{V_{bi}}}.$$

順バイアスで狭く、逆バイアスで広くなる。空間電荷量が電圧で変化するので接合は容量として働き、単位面積あたり

$$C = \frac{dQ}{dV} = \frac{\varepsilon_s}{W} = \sqrt{\frac{e\varepsilon_s N_D}{2(V_{bi}-V)}}\quad(\text{片側接合}).$$

$1/C^2$ を $V$ に対してプロットすると直線になり、傾きからドーピング密度、切片から $V_{bi}$ が求まる。

---

## 6. I–V 特性（ショックレー方程式）

### 仮定

1. 急峻な空乏層近似（外部電圧は空乏層に印加、外側は中性）。
2. ボルツマン近似。
3. **低注入**（注入された少数キャリア密度 < 多数キャリア密度）。
4. 空乏層内でのキャリア生成・再結合はない。

### 空乏層端での少数キャリア密度

$V=0$（熱平衡）では、$p$ 型層の $E_C$ 以上のエネルギーを持つ電子数が両側で等しく、

$$n_{p0} = n_{n0}\exp\!\left[-\frac{eV_{bi}}{k_BT}\right].$$

$V\ne0$ では障壁が $e(V_{bi}-V)$ に下がり、$p$ 側空乏層端 $x=x_p$ での電子密度は

$$n_p(x_p) = n_{n0}\exp\!\left[-\frac{e(V_{bi}-V)}{k_BT}\right] = n_{p0}\exp\!\left[\frac{eV}{k_BT}\right].$$

順バイアスで少数キャリア（電子）が $p$ 側へ**注入**される（過剰注入）。

### 少数キャリアの拡散と電流

$p$ 中性領域では、注入電子が再結合しながら拡散して平衡値 $n_{p0}$ に戻る。定常拡散方程式

$$0 = -\frac{n_p-n_{p0}}{\tau_e} + D_e\frac{d^2 n_p}{dx^2}$$

を境界条件 $n_p(x_p)=n_{p0}e^{eV/k_BT}$、$n_p(\infty)=n_{p0}$ で解くと

$$n_p(x) = n_{p0} + n_{p0}\bigl(e^{eV/k_BT}-1\bigr)\exp\!\left[-\frac{x-x_p}{L_e}\right],
\qquad L_e = \sqrt{D_e\tau_e}\ (\text{拡散長}).$$

これから電子の拡散電流密度が求まり、正孔についても同様に計算して和をとると、**ショックレー（理想ダイオード）方程式**

$$\boxed{\,J = J_0\left[\exp\!\left(\frac{eV}{k_BT}\right)-1\right],\qquad
J_0 = e\left(\frac{D_e n_{p0}}{L_e} + \frac{D_h p_{n0}}{L_h}\right)\,}$$

を得る。電流は **少数キャリア**によって支配される。$J_0$ は逆方向飽和電流密度。

---

## 7. キャリアの生成と再結合

- **生成**：熱や光で価電子帯電子が伝導帯へ励起。熱的生成率 $G(T)$ は温度で決まる。
- **再結合**：
  1. バンド間再結合（電子–正孔対、フォトン放出やオージェ）
  2. バンドギャップ内の**再結合中心**を介した再結合

再結合速度は電子・正孔密度に比例（$U\propto np$）。熱平衡では $G(T)=U n_0 p_0$ が釣り合う。

### 空乏層内再結合電流

理想式は空乏層内再結合を無視するが、実際には $p\simeq n$ となる空乏層で再結合電流が流れる。再結合率を $U$、再結合寿命を $\tau=1/(N_r\sigma v_{th})$ とすると

$$J_r \simeq eU(x_n+x_p) = \frac{e n_i}{2\tau}(x_n+x_p)\exp\!\left[\frac{eV}{2k_BT}\right].$$

拡散電流（$\propto e^{eV/k_BT}$）と再結合電流（$\propto e^{eV/2k_BT}$）を合わせると、一般には

$$J \propto \exp\!\left(\frac{eV}{n\,k_BT}\right)$$

と書け、**理想係数（ideality factor）**$n$ は拡散支配で $1$、再結合支配で $2$ に近づく。

---

## 8. 降伏特性（逆バイアス）

逆バイアスを強くすると、ある電圧で急激に電流が流れる**降伏（breakdown）**が起こる。

- **なだれ（avalanche）降伏**：高電界で加速された電子が価電子帯電子を伝導帯へたたき上げる**衝突電離**を繰り返し、キャリアが指数的に増える。
- **ツェナー（Zener）効果／トンネル降伏**：不純物濃度が高い接合で、逆バイアスの強電界により価電子帯の電子がトンネル効果で伝導帯へすり抜ける。

---

## 補足：少数キャリアの拡散方程式

電子・正孔の電流はドリフト＋拡散で

$$J_n = ne\mu_e\mathcal E + eD_e\frac{\partial n}{\partial x},\qquad
J_p = pe\mu_h\mathcal E - eD_h\frac{\partial p}{\partial x}.$$

連続の方程式 $\partial n/\partial t = G_e - U_e + \tfrac1e\nabla\!\cdot J_n$ に代入し、定常・中性領域（$\mathcal E=0$）・生成なしとすると、消滅率 $U = (p-p_0)/\tau$ を用いて

$$0 = -\frac{p-p_0}{\tau_h} + D_h\frac{d^2 p}{dx^2},\qquad
0 = -\frac{n-n_0}{\tau_e} + D_e\frac{d^2 n}{dx^2}$$

が得られる。これが I–V 特性導出で用いた拡散方程式である。

---

## キーポイント

- ビルトイン電位 $eV_{bi}=k_BT\ln(N_D N_A/n_i^2)$。
- 空乏層幅 $W\propto\sqrt{(V_{bi}-V)}$、接合容量 $C=\varepsilon_s/W$。$1/C^2$–$V$ プロットで $N$ と $V_{bi}$ が求まる。
- 理想ダイオード：$J=J_0(e^{eV/k_BT}-1)$、少数キャリア支配。
- 空乏層内再結合で理想係数 $n$ は $1$（拡散）〜$2$（再結合）。
- 逆方向降伏はなだれ（衝突電離）とツェナー（トンネル）。
