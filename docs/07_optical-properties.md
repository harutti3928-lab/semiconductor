# 第7回 半導体の光学的性質

## この回の内容

- 半導体の光吸収：波長依存性、吸収係数（直接許容/禁制/間接遷移）、電子と光の相互作用、自由キャリア吸収、励起子吸収、量子井戸の光吸収
- 半導体の発光現象：励起方法による分類、放射/非放射再結合、励起子発光、ドナー–アクセプタ対発光

---

## 1. 光吸収の波長依存性

半導体の吸収機構をエネルギーの低い順に並べると：

| 機構 | エネルギー領域 |
|------|----------------|
| 自由キャリア（電子・正孔）吸収 | マイクロ波 |
| 不純物吸収（ドナー・アクセプタ） | 遠赤外 |
| 格子振動（光学フォノン）吸収 | 赤外 |
| **基礎吸収**（価電子帯→伝導帯遷移） | 可視〜近赤外 |
| 励起子吸収 | 基礎吸収端近傍 |

基礎吸収の起こる最低エネルギーを**基礎吸収端**という。

---

## 2. 吸収係数と侵入長

エネルギー流密度 $I$ の減衰から吸収係数 $\alpha$ を定義する。

$$\alpha(\omega) = -\frac{1}{I(\omega)}\frac{dI(\omega)}{dz}
\quad\Longrightarrow\quad
I(z) = I_0\,e^{-\alpha z}.$$

**侵入長**は $1/\alpha$。Si（300 K）の例：Ar レーザ (488 nm) で $\alpha=10^4\ \mathrm{cm^{-1}}$、$1/\alpha=1\ \mu\mathrm m$。Nd:YAG (1064 nm) では $\alpha=20\ \mathrm{cm^{-1}}$、$1/\alpha=500\ \mu\mathrm m$（間接遷移半導体はバンドギャップ近傍で吸収が弱く深く侵入する）。

吸収係数は、始状態（密度 $n_i$）から終状態（密度 $n_f$）への遷移確率 $W_{if}$ を用いて

$$\alpha(\omega) = A\,W_{if}\,n_i\,n_f\quad(\text{和は }E_f-E_i=\text{一定でとる}).$$

---

## 3. 直接許容遷移

放物線バンド（直接型）でエネルギー保存 $E_f-E_i=\hbar\omega$、運動量保存を課す。

$$E_f = \frac{\hbar^2 k^2}{2m_e^*}+E_g,\quad E_i = -\frac{\hbar^2 k^2}{2m_h^*}
\;\Rightarrow\;
\hbar\omega - E_g = \frac{\hbar^2 k^2}{2m_r},\quad \frac1{m_r}=\frac1{m_e^*}+\frac1{m_h^*}.$$

$m_r$ は換算質量。**結合状態密度**は自由電子の状態密度と同型で

$$N(\hbar\omega) = \frac{(2m_r)^{3/2}}{2\pi^2\hbar^3}(\hbar\omega-E_g)^{1/2}.$$

遷移確率が光エネルギーによらないので、吸収係数は

$$\boxed{\,\alpha(\omega) = A^*(\hbar\omega - E_g)^{1/2}\,}
\qquad
A^* \simeq \frac{e^2}{n c\hbar^2 m_e^*}\left(\frac{m_e^* m_h^*}{m_h^*+m_e^*}\right)^{3/2}.$$

屈折率 $n=4$、自由電子質量とすると $\alpha\simeq2\times10^4(\hbar\omega-E_g)^{1/2}\ \mathrm{cm^{-1}}$（$\hbar\omega$ は eV）。$\alpha^2$–$\hbar\omega$ プロットの外挿から $E_g$ が求まる。

**直接禁制遷移**（$k=0$ で行列要素が消える場合）は $k^2$ に比例して遷移確率が増えるので

$$\alpha(\omega) = A(\hbar\omega - E_g)^{3/2}.$$

---

## 4. 間接遷移

始状態と終状態の波数が異なる（$k_i\ne k_f$）ため、運動量保存を**フォノン**（波数 $\mathbf p$、エネルギー $E_p$）が担う。エネルギー保存は

$$E_f - E_i \pm E_p = \hbar\omega\quad(\text{フォノン吸収}+/\text{放出}-).$$

吸収係数は（始状態密度）×（終状態密度）×（フォノン数）を可能な全状態で積分して得られ、フォノン数はボーズ統計に従う。

$$\alpha_a(\omega) = \frac{A(\hbar\omega-E_g+E_p)^2}{\exp(E_p/k_BT)-1}\quad(\text{フォノン吸収},\ \hbar\omega>E_g-E_p),$$

$$\alpha_e(\omega) = \frac{A(\hbar\omega-E_g-E_p)^2}{1-\exp(-E_p/k_BT)}\quad(\text{フォノン放出},\ \hbar\omega>E_g+E_p).$$

$\hbar\omega>E_g+E_p$ では両方可能で $\alpha=\alpha_a+\alpha_e$。**$\alpha^{1/2}$–$\hbar\omega$ プロットが2本の直線**になり、その折れ曲がりから $E_g\pm E_p$、したがって $E_g$ とフォノンエネルギー $E_p$ が求まる。吸収の温度依存性はフォノン数（ボーズ因子）を通して現れる。

---

## 5. 吸収端に関する現象

- **Burstein–Moss シフト**：縮退半導体でバンドが電子に埋められる（band filling）と、価電子帯から占有済み準位への遷移が禁じられ、実効的な吸収端が高エネルギー側にシフトする。
- **アーバック則（Urbach's rule）**：吸収端が指数関数的に立ち上がり、$d(\ln\alpha)/d(\hbar\omega)=1/k_BT$。バンドの裾（不純物バンド）や励起子に由来する。

---

## 6. 電子と光の相互作用

### 半古典論：遷移双極子モーメント

光の電場を $E_x=E_0 e^{i\omega t}$（波長 ≫ 原子サイズ）とすると、摂動ハミルトニアン $H'=exE_0 e^{i\omega t}$。状態 $n\to m$ の遷移確率は

$$W_{mn} = \frac{2\pi}{\hbar}\bigl|\langle\varphi_m|H'|\varphi_n\rangle\bigr|^2,\qquad
M_{mn} = \langle\varphi_m|ex|\varphi_n\rangle\ (\text{遷移双極子モーメント}).$$

**電気双極子遷移の選択則**（パリティ）：

- 許容遷移：始状態と終状態が**異なるパリティ**（s–p, d–f など）。
- 禁制遷移：**同じパリティ**（s–d, p–p など）。

### 量子論：$\mathbf p\cdot\mathbf A$ 相互作用

電磁場をベクトルポテンシャル $\mathbf A$ で表すと、ハミルトニアンは $H=\frac{1}{2m}(\mathbf p+e\mathbf A)^2+V$ で、摂動項 $H'=\frac{e}{m}\mathbf p\cdot\mathbf A$。$\mathbf A$ を生成消滅演算子で書き、価電子帯ブロッホ状態（波数 $k'$）→伝導帯（波数 $k$）の遷移を計算すると、光の波数 $k_p\simeq0$ より

$$\boxed{\,k = k'\quad(\text{直接遷移})\,}$$

同じ波数状態間でのみ光学遷移が可能。行列要素 $G_1$ が消えない条件から、$u_c$ と $u_v$ が異なるパリティを持つとき許容遷移となり、$\alpha\propto(\hbar\omega-E_g)^{1/2}$ が再現される。放射場の量子化では、光子数 $n$ の状態に対し吸収 $\propto n$、放出 $\propto(n+1)$（自然放出 + 誘導放出、→ 第8回レーザ）。

## 7. 励起子（exciton）

光吸収で生成した伝導帯電子（$-e$）と価電子帯正孔（$+e$）がクーロン引力で束縛状態を作ったもの。ボーズ統計に従う中性粒子で、電気伝導には寄与しないが光学的性質に大きく関与する。

- **自由励起子（モット・ワニエ励起子）**：束縛が緩く広がった対。半導体で観測。
- **局在励起子（フレンケル励起子）**：構成原子に強く局在。イオン結晶・分子結晶。

自由励起子は水素様原子モデルで扱える。全エネルギーは重心運動 $E_K$ と相対運動 $E_n$ の和：

$$E = E_K + E_n,\qquad E_K = \frac{\hbar^2 K^2}{2M},\ M=m_e^*+m_h^*,$$

$$E_n = -13.6\ \mathrm{eV}\cdot\frac{\mu/m}{\varepsilon_r^2}\frac{1}{n^2},\qquad
\frac1\mu = \frac1{m_e^*}+\frac1{m_h^*},$$

$$a_B^* = a_B\,\frac{m}{\mu}\,\varepsilon_r = 0.053\ \mathrm{nm}\cdot\frac{m}{\mu}\varepsilon_r.$$

励起子準位は伝導帯直下 $n=1,2,\dots$ に現れ、価電子帯頂上からこれらへの光学遷移が吸収スペクトルに鋭い線を与える。GaAs の励起子束縛エネルギーは約 4.2 meV と小さく、低温で顕著。

---

## 8. 量子井戸の光吸収

量子井戸では電子・正孔がサブバンドを作り、遷移エネルギーは

$$E(n,k_x,k_y) = E_G + E_e(n_e) + E_h(n_h) + \frac{\hbar^2}{2m_r}(k_x^2+k_y^2).$$

$z$ 方向包絡関数の重なり $\int\zeta_e\zeta_h\,dz$ から**選択則 $\Delta n = n_e - n_h = 0$** が出る（主に同じ量子数間の遷移）。2次元の結合状態密度が階段状なので、吸収スペクトルは**階段状**になり、各ステップの立ち上がりに励起子ピークが重なる。井戸幅で吸収端を制御できる。

---

## 9. 発光現象

### 励起方法による分類

| 名称 | 励起源 |
|------|--------|
| 熱発光（黒体放射） | 加熱 |
| フォトルミネセンス (PL) | 光照射 |
| カソードルミネセンス (CL) | 電子線 |
| エレクトロルミネセンス (EL) | 電圧印加 |
| 化学発光／生物発光／摩擦発光 | 化学反応など |

### 放射再結合と非放射再結合

励起された電子は一定寿命で正孔と再結合する。

- **放射再結合**（発光）：寿命 $\tau_r$
- **非放射再結合**（多フォノン放出、オージェ過程、欠陥準位）：寿命 $\tau_{nr}$

過剰キャリア密度 $\Delta n$ に対し、放射・非放射の回数は $R_r=\Delta n/\tau_r$、$R_{nr}=\Delta n/\tau_{nr}$。**発光効率**は

$$\boxed{\,\eta = \frac{R_r}{R_r+R_{nr}} = \frac{1}{1+\tau_r/\tau_{nr}}\,},\qquad
\frac{1}{\tau_{\text{eff}}} = \frac{1}{\tau_r}+\frac{1}{\tau_{nr}}.$$

弱励起（$\Delta n=\Delta p\ll n_0,p_0$）での放射再結合寿命は

$$\tau_r = \frac{\Delta n}{R} = \frac{n_0 p_0}{(n_0+p_0)R_0}.$$

真性半導体では $\tau_r$ が長く、不純物半導体では短い。

### 半導体からの発光過程

(a) 自由電子–自由正孔、(b) 自由電子–アクセプタ束縛正孔（およびその逆）、(c) 自由励起子、(d) 中性ドナー束縛励起子、(e) ドナー–アクセプタ対発光。実際には不純物に束縛された励起子からの発光が自由励起子より強く観測されることが多い。

自由励起子の発光エネルギーは（運動エネルギーがほぼ 0 なので）

$$E_{\text{ex}} = \hbar\nu_{\text{ex}} = E_G - |E_n|,$$

バンドギャップの温度依存性は **Varshni の経験式** $E_g(T)=E_g(0)-\alpha T^2/(T+\beta)$ に従う。

---

## 10. ドナー–アクセプタ対（D–A ペア）発光

ドナーに捕らえられた電子とアクセプタに捕らえられた正孔の再結合。発光前は両者中性、発光後は両者イオン化してクーロンエネルギーの利得が生じるため、対の距離 $r$ に依存して発光エネルギーが変わる。

$$\boxed{\,E(r) = \hbar\nu(r) = E_G - (E_D + E_A) + \frac{e^2}{4\pi\varepsilon_0\varepsilon_r r}\,}$$

- 遷移確率は両者の波動関数の重なりに依存し、近い対ほど高い。
- 励起強度を強くすると、遠い対（遷移確率が低く飽和しやすい）が飽和するため、発光ピークが**高エネルギー側にシフト**する。
- 結晶格子上で許される離散的な $r$ に対応して、スペクトルが離散線の列になる。

---

## キーポイント

- 吸収端の形で遷移型が判別できる：直接許容 $\alpha\propto(\hbar\omega-E_g)^{1/2}$、直接禁制 $\propto{}^{3/2}$、間接 $\propto(\hbar\omega-E_g\pm E_p)^2$。
- 直接遷移は $k=k'$、電気双極子選択則はパリティで決まる。
- 励起子は水素様モデル、$E_n\propto\mu/\varepsilon_r^2$。量子井戸吸収は階段状＋選択則 $\Delta n=0$。
- 発光効率 $\eta=1/(1+\tau_r/\tau_{nr})$。
- D–A ペア発光は距離依存 $E(r)=E_G-(E_D+E_A)+e^2/4\pi\varepsilon_0\varepsilon_r r$。
